About SBPalette

SBPalette is a tool for finding the closest armor colors in Hypixel SkyBlock. The application uses an advanced color comparison algorithm (delta E CIE76) to find the most similar armor colors to your target color.

üåü Open Source Project

SBPalette is an open-source project! You can contribute, report issues, or suggest improvements on our GitHub repository. We welcome all contributions from the community.

Key Features

  • Search by HEX color code
  • Display of 5 closest matches
  • Search history
  • 5 beautiful themes
  • Ability to share colors via link
  • XML query support for Google Sheets integration
  • Copy table rows for Excel or Google Sheets

How to Use

1. Color Input

Enter the HEX color code in the input field (example: FF0000 or #FF0000)

2. Search

Click the "Find" button or press Enter to search

3. Results

View the table with 5 closest matches. Each match contains:

  • Armor name
  • HEX color code
  • Visual color display
  • Color difference
  • Match rank (T0-T4)
